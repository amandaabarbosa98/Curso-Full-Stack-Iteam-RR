# Curso Full Stack Developer - Iteam RR

Este repositório documenta minha jornada no **Curso Full Stack Developer**, uma formação completa que abrange desde lógica de programação até DevOps e Cloud Computing.

O curso tem como objetivo capacitar o aluno para atuar no desenvolvimento **Front-end, Back-end, Mobile, Banco de Dados, Testes e Infraestrutura**, proporcionando uma visão completa do ciclo de vida de desenvolvimento de software.

---

## 📚 Estrutura do Curso

O curso é dividido em 11 módulos:

---

## 🧠 Módulo 01 – Lógica de Programação e Algoritmos (32h)

- Conceitos básicos de lógica, algoritmo e programação  
- Construção de algoritmos (narrativa, fluxograma e pseudocódigo)  
- Variáveis e tipos de dados  
- Estruturas condicionais e de repetição  
- Implementação prática com **Python**  
- Desenvolvimento de pequenos projetos

---

## 🗂️ Módulo 02 – Arquitetura da Informação (32h)

- Fundamentos da arquitetura da informação  
- Organização e estruturação de conteúdo  
- Taxonomias e metadados  
- Sistemas de busca  
- Design centrado no usuário  
- Prototipação e testes de usabilidade  
- Acessibilidade e design inclusivo  

---

## 🌐 Módulo 03 – Fundamentos de Desenvolvimento Web (32h)

- Conceitos de internet e modelo cliente-servidor  
- HTML (estrutura, semântica, formulários, multimídia)  
- CSS (box model, flexbox, grid, responsividade)  
- JavaScript (DOM, eventos, lógica e interatividade)  
- Desenvolvimento de páginas web responsivas  

---

## ⚛️ Módulo 04 – Desenvolvimento Front-end com React (40h)

- Componentes, JSX, props e state  
- Hooks (useState, useEffect, useContext)  
- React Router  
- Consumo de APIs (fetch / Axios)  
- Gerenciamento de estado  
- Construção de aplicações SPA  

---

## 🖥️ Módulo 05 – Desenvolvimento Back-end com Node.js e Express (40h)

- Fundamentos do Node.js  
- Criação de APIs com Express  
- Middlewares  
- Integração com banco de dados (MongoDB ou PostgreSQL)  
- CRUD completo  
- Autenticação e autorização  
- Testes e deploy  

---

## 📱 Módulo 06 – Desenvolvimento Mobile com React Native (32h)

- Estrutura de aplicações mobile  
- Componentes nativos  
- Navegação (Stack, Tabs, Drawer)  
- Consumo de APIs  
- Armazenamento local  
- Publicação na Play Store e App Store  

---

## 🐍 Módulo 07 – Programação em Python (40h)

- Fundamentos da linguagem  
- Estruturas de controle  
- Estruturas de dados  
- Funções e módulos  
- Programação Orientada a Objetos  

---

## ⚙️ Módulo 08 – Programação em C++ (40h)

- Conceitos básicos e sintaxe  
- Estruturas de controle  
- Funções e ponteiros  
- Programação Orientada a Objetos  
- STL e recursos avançados  

---

## 🗄️ Módulo 09 – Bancos de Dados e SQL (32h)

- Modelo relacional  
- Normalização  
- Comandos SQL (DDL e DML)  
- JOINs e subconsultas  
- Transações, views e triggers  
- Projeto prático de banco de dados  

---

## 🧪 Módulo 10 – Testes e Qualidade de Software (20h)

- Conceitos de qualidade  
- Testes de unidade, integração e sistema  
- Testes funcionais e não funcionais  
- Automação de testes  
- CI/CD  

---

## ☁️ Módulo 11 – DevOps e Cloud Computing (32h)

- Cultura DevOps  
- CI/CD  
- Docker e Kubernetes  
- Infraestrutura como Código  
- Cloud Computing (AWS, Azure, GCP)  
- Deploy de aplicações na nuvem  

---

## 🎯 Objetivo da Formação

Ao final do curso, o aluno estará apto a:

- Desenvolver aplicações web completas (Front-end e Back-end)
- Criar aplicações mobile multiplataforma
- Projetar e modelar bancos de dados
- Implementar testes automatizados
- Realizar deploy em ambientes cloud
- Trabalhar com práticas modernas de DevOps

---

## 🛠️ Tecnologias Abordadas

- HTML, CSS, JavaScript
- React e React Native
- Node.js e Express
- Python e C++
- SQL, MongoDB, PostgreSQL
- Docker, Kubernetes
- Git e CI/CD

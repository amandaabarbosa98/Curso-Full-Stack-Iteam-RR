# ⚙️ Programação em C++

Este repositório reúne os estudos, exercícios e projetos desenvolvidos durante o **Módulo 08 – Linguagem de Programação C++** do curso Full Stack.

O objetivo deste módulo é fornecer uma base sólida na linguagem C++, abordando desde os conceitos fundamentais até recursos avançados da linguagem, incluindo Programação Orientada a Objetos, ponteiros, templates e a STL (Standard Template Library).

---

## 🎯 Objetivos

- Compreender os fundamentos da linguagem C++;
- Desenvolver programas utilizando estruturas de controle e funções;
- Trabalhar com ponteiros e alocação dinâmica de memória;
- Aplicar conceitos de Programação Orientada a Objetos;
- Utilizar recursos avançados da linguagem e bibliotecas padrão;
- Desenvolver soluções eficientes para problemas computacionais.

---

## 📚 Conteúdos Estudados

### 🚀 Introdução ao C++

- História e evolução do C++;
- Configuração do ambiente de desenvolvimento;
- Variáveis e tipos de dados;
- Operadores e expressões;
- Entrada e saída de dados com `iostream`;
- Comentários e documentação de código.

### 🔄 Estruturas de Controle

- Condicionais:
  - `if`
  - `else`
  - `switch`

- Estruturas de repetição:
  - `for`
  - `while`
  - `do-while`

- Controle de fluxo:
  - `break`
  - `continue`
  - `goto`

- Tratamento de exceções:
  - `try`
  - `catch`
  - `throw`

### 🔧 Funções e Ponteiros

- Definição e chamada de funções;
- Parâmetros e retorno de valores;
- Funções recursivas;
- Passagem por valor e por referência;
- Ponteiros;
- Aritmética de ponteiros;
- Alocação dinâmica de memória.

### 🏗️ Programação Orientada a Objetos

- Classes e objetos;
- Atributos e métodos;
- Construtores e destrutores;
- Encapsulamento;
- Herança;
- Polimorfismo;
- Sobrecarga de operadores;
- Templates.

### 📦 Bibliotecas e Recursos Avançados

- STL (Standard Template Library):
  - Vetores (`vector`)
  - Filas (`queue`)
  - Pilhas (`stack`)
  - Mapas (`map`)
  - Conjuntos (`set`)

- Algoritmos da STL;
- Iteradores;
- Manipulação de arquivos (`fstream`);
- Manipulação de strings (`string`);
- Namespaces;
- Expressões lambda;
- Smart pointers.
